# Usage

Add script after webslides.js and call `addWebSlideAnimation`

```
<script src="./webslides.js"></script>
<script src="./webslides-animation.js"></script>
<script>
  window.ws = new WebSlides();
  addWebSlideAnimation(window.ws);
</script>
```
